# Contributing

Thank you for your interest in LAO!

## Bug Reports & Feature Requests

Issues are welcome. Please use [GitHub Issues](../../issues) to report bugs or suggest features.

When reporting a bug, include:
- macOS version
- Xcode version
- Steps to reproduce
- Expected vs actual behavior

## Pull Requests

We are not accepting pull requests at this time. This may change in the future.

## Questions

For general questions, please open a [GitHub Discussion](../../discussions) or an issue.
