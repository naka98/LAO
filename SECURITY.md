# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue.**

Instead, report it privately via [GitHub Security Advisories](../../security/advisories/new) or contact [@naka98](https://github.com/naka98) directly.

We will acknowledge receipt within 72 hours and provide an update within 7 days.

## Scope

This policy applies to the LAO macOS application and its associated packages in this repository.
